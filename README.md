但话题又回到，记忆回闪→焦虑→记忆回闪，为什么是先有记忆回闪，而不是直接焦虑然后记忆回闪，这很容易理解，第一遗传，第二关联思维。那么我先讲第二个。关联思维是随机关联，准确的说我的思维就像网一样四通八达，想一件事我可以得出更多结论，那如果将记忆一起变得像网一样，这就能解释为什么我一直出现记忆回闪是完全随机的，但这远远不够加上我的感官敏感，很容易记住与一件事不相干的，比如空间信息物品摆放，这些都与感官敏感有直接关系，过目不忘与感官敏感有一定关系，人的记忆形式是通过感官记忆，如果一个人感官敏感那么很容易记录大量信息，记忆回闪的“事”是灵感的另一种表现。
